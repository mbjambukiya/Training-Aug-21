Function Closure

Closure in JavaScript is function along with its lexical scope.It means function is bundeled with its scope.

For example, if we have local variable inside function fun1 and we have another function fun2 which is closure to fun1 and fun2 has access to that variable and fun2 has reference of that variable instead of value. Now we can access that local variable of fun1 outside of fun1 using reference of that variable using fun2.

Example:

function fun1(){
var x=5;
function fun2(){
x+=5;
alert(x);
}
return fun2;
}
var p = fun1();
var y = p();
y = p();
y = p();

Value of x now : y
