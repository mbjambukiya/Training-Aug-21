Function Hoisting

 JavaScript puts all the declaration to the top of current scope. It means internally declaration is at top of the scope for declaration done in anywhere in script.so that variables and functions can be used in anywhere in the current scope even before declaration.

For example, if we declare variable or function at the end of script, still we can use it in anywhere in current scope.

Example:

getSquare(5);
function getSquare(number){
return number*number;
}
